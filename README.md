# Hydro Nails and Spa — Redesign Concept

An unsolicited, single-page website redesign concept for **Hydro Nails and Spa**, an
appointment-first nail studio in Prairieville, Louisiana. This is a "this could be yours"
pitch piece, not the official site.

## Why a redesign?

The business currently has **no real website** — its only web presence is a Facebook page
(100% recommended across ~37 reviews) plus scattered directory listings. That means:

- No home online for their strong reputation, service menu, or gallery.
- No self-serve way for guests to request an appointment.
- Hours, address, and services live in different (and slightly inconsistent) places across
  Yelp, Nextdoor, and directory sites.

This concept gives them a fast, elegant, mobile-first home that shows off the work and makes
requesting an appointment effortless.

## What's inside

- Sticky shrinking header, smooth-scroll nav, scroll-reveal sections, subtle hero motion
  (respects `prefers-reduced-motion`).
- Service menu (manicures, dip powder, spa pedicures, acrylic/gel extensions, nail art, removal).
- Nail-art gallery (with a CDN lightbox ready for real photos).
- Real reviews, at-a-glance stats, hours, address, click-to-call, and an appointment-request form.
- Fully static: `index.html` + `styles.css` + `script.js`. No build step.

## Design

A refined, "hydro"-inspired palette — deep teal + soft aqua + warm blush/champagne on cream —
deliberately avoiding the dated pink nail-salon template. Type pairing: Fraunces (display serif)
+ Jost (clean sans).

## Real data vs. placeholders

- **Real:** business name, address (17900 Airline Hwy, Ste E, Prairieville, LA 70769), phone
  (225) 677-0507, appointment-first model, services (dip/gel/mani/pedi/acrylic/art), reviews,
  and the 100%-recommended / 37-reviews social proof.
- **Not published anywhere public → not fabricated:** prices (shown as "Call for pricing",
  marked `TODO`) and exact current hours (marked `TODO`; hours vary slightly across listings).
- **Images:** the salon's real photos live on Facebook/Yelp, which are token/access-blocked.
  Tasteful CSS/SVG placeholders are used and marked `IMG-NEEDED`. See
  [`assets/photos/DROP-PHOTOS-HERE.md`](assets/photos/DROP-PHOTOS-HERE.md) to add real ones.

Open `index.html` in any browser — no server needed.
